About Poet 2.0

Poet 2.0 is a Nelson Public Libraries project created as part of the New Zealand Poetry day celebrations.

Poet 2.0 is based on a modified and extended version of Zack Schollz's Poem generator and incorporates the NodeBox linguistic module.

This application has been created by Steve Nickalls (Senior Library Assistant, Nelson Public Libraries). It is freely available and open source

Zacks GitHub: https://github.com/schollz/poetry-generator

NodeBox: https://www.nodebox.net/code/index.php/Home

Steve’s GitHub: https://github.com/snickalls/poet2.0

****If reading this before August 2018 please note that this application is still under development. Iyt is constantly changing and being added to. SO don't judge the spaghetti code!****
